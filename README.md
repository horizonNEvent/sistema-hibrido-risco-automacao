# O Futuro do Mercado de Trabalho com Inteligência Artificial
**Análise Multidimensional: Decisão, Impacto e Ferramentas** [1]

## 📌 Sobre o Projeto
Este repositório contém o código-fonte e a pesquisa desenvolvidos para avaliar como a Inteligência Artificial está transformando o mercado de trabalho nos diferentes níveis de tomada de decisão (operacional, tático e estratégico) e quais são os impactos mensuráveis para o cenário brasileiro e global até 2030 [4].

O núcleo técnico deste projeto é uma **Arquitetura Neuro-Simbólica** [2]. Trata-se da integração inédita de dois modelos de Inteligência Artificial:
1. **Cérebro Estatístico (Rede Neural Artificial):** Um modelo preditivo construído em TensorFlow/Keras que mapeia o risco de automação de ocupações com base em características comportamentais, atingindo **98% de acurácia** no conjunto de testes [5, 6].
2. **Cérebro Lógico (Sistema Especialista):** Um motor de inferência baseado na biblioteca *Experta* que aplica regras lógicas estritas para garantir a explicabilidade (XAI) do *porquê* uma profissão possui Risco Alto, Médio ou Baixo [7, 8].

## 📊 Fontes de Dados
A modelagem e os insights presentes neste repositório foram fundamentados no cruzamento de dados de fontes primárias de excelência [3]:
* **Fórum Econômico Mundial (WEF):** *The Future of Jobs Report 2025*
* **IBGE:** Pesquisa de Inovação (PINTEC) com foco no uso de tecnologias digitais avançadas na indústria nacional.
* **FGV IBRE:** Dados sobre o impacto da tecnologia no emprego formal.
* **Kaggle:** Dataset estruturado *"AI Job Market Insights"*.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Python 3.10+ [9]
* **Machine Learning:** TensorFlow, Keras, Scikit-learn [10, 11]
* **Sistemas Simbólicos:** Experta, Scikit-fuzzy [11]
* **Análise e Visualização (Tech Noir):** Pandas, NumPy, Matplotlib, Seaborn [10, 12]
* **Engenharia de Prompt:** Técnicas de *Chain-of-Thought*, *Few-Shot* e *Role-Playing* aplicadas via LLMs [13, 14].
